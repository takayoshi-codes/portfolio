Business Automation — 08
競合価格モニタリング＆通知 Bot

EC サイトの競合商品価格を定期スクレイピングし、変動があれば Slack またはメールで自動通知。GitHub Actions で Cron 実行し常時監視を無人化。config.yaml 一行で通知先を切替可能。

使用技術
BeautifulSoupSQLite Slack WebhookGmail API GitHub Actions
切替設計
NOTIFY_TYPE: slack
# または email
主な機能
価格取得・差分検知
Slack ／ メール通知
履歴 DB 保存
日次サマリーレポート
価格モニタリングデモ
通知先:
商品前回現在変動
