Web App — 01
在庫・発注管理 Web アプリ

商品の在庫数・発注状況をリアルタイムで管理。在庫が閾値を下回ると自動アラートを表示し、発注ボタン一つで履歴を記録。Django + PostgreSQL でフルスクラッチ実装。

使用技術
DjangoPostgreSQL BootstrapRailway
主な機能
商品 CRUD・在庫アラート
発注履歴・ログイン認証
管理画面・DB リレーション設計
アピール点
DB 設計・認証・デプロイまで一貫実装。実務レベルの Web アプリ構築力を提示。
